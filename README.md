# Yong Yu Academic Homepage

Static academic homepage designed for GitHub Pages.

## Local Preview

Open `index.html` directly in a browser, or serve the folder:

```powershell
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## GitHub Pages

For a personal GitHub Pages site, publish this folder to a public repository named:

```text
YongYU-PKU2023.github.io
```

GitHub will serve it at:

```text
https://yongyu-pku2023.github.io/
```
